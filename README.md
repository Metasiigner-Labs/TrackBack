# TrackBack

**They track us. We track them back.**

TrackBack is a public accountability website showing how much special interest, lobbyist, and corporate money each member of Congress takes — using **real, legally obtained government data**.

## Data Sources (100% Legal & Public)

| Source | Data Used | License / Access |
|--------|-----------|------------------|
| [FEC Bulk Data](https://www.fec.gov/data/browse-data/?tab=bulk-data) | candidate_summary, pas2, indiv, ccl, independent expenditures | Public government data |
| [GovTrack.us](https://www.govtrack.us/) | Congressional roll call votes | Open API |
| [unitedstates/congress-legislators](https://github.com/unitedstates/congress-legislators) | Current member roster, bioguide IDs | Public domain |
| [Congress.gov Bioguide](https://bioguide.congress.gov/) | Official member photos | Public domain (U.S. government) |
| [U.S. Census Geocoder](https://geocoding.geo.census.gov/) | Zip code → congressional district | Public government API |
| [Zippopotam.us](https://zippopotam.us/) | Zip code → coordinates | Free geocoding API |

We also link to [OpenSecrets](https://www.opensecrets.org/) profiles for additional context (external site).

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Sync real congressional data

```bash
npm run sync-data
```

This fetches all ~539 current members from FEC bulk files and GovTrack. First run downloads ~4GB of individual contribution data (`indiv24.zip`) — allow 30–90 minutes. Cached runs take ~2–5 minutes.

To skip the large individual-contributions file (faster, less complete):

```bash
# PowerShell
$env:SKIP_INDIV_SYNC="1"; npm run sync-data
```

### 3. Run the site

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Purity Score

Calculated transparently from FEC filings:

1. **Base score** = 100 − % of receipts from non-individual sources (PACs, party committees, transfers)
2. **Bonuses** (+10 to +20) for Nay votes on bills touching top donor industries (GovTrack)
3. **Penalties** (−5 to −30) for high PAC dependence and controversial-industry donors

## Refreshing Data

Re-run `npm run sync-data` periodically (weekly/monthly) to pick up new FEC filings and votes. The site reads from the generated JSON cache for fast page loads.

## Deployment (Vercel — free, global CDN)

1. Push this repo to GitHub
2. Import at [vercel.com/new](https://vercel.com/new)
3. Run `npm run sync-data` locally first (commits `src/data/politicians.json`) **or** add a build step — recommended: sync locally, commit JSON, deploy static site
4. Build command: `npm run build` — Output: Next.js default

```bash
npm run sync-data
npm run build
npx vercel --prod
```

The site is fully static after build — no runtime API keys required.

### What AIPAC means on TrackBack

AIPAC is a **lobbying organization**, not a campaign committee — it does not appear as a direct donor in FEC filings. Related influence shows up as:

- **United Democracy Project** and other pro-Israel super PACs (independent expenditures)
- Affiliated PAC contributions (pas2 bulk data)
- Future: LDA lobbying disclosures and bundled individual giving

We link to [OpenSecrets AIPAC profile](https://www.opensecrets.org/orgs/american-israel-public-affairs-cmte/summary) for supplemental research.
